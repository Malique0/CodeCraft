Über das Spiel

In diesem Spiel wirst du Schritt für Schritt lernen, wie du mit Spring Boot programmierst. Das Ziel des Spiels ist es, verschiedene Aufgaben und Herausforderungen zu meistern, die dir helfen, wichtige Konzepte von Spring Boot zu verstehen und anzuwenden.

Du wirst Aufgaben lösen, bei denen du REST-APIs erstellen, Datenbanken integrieren, Fehler beheben und vieles mehr lernen wirst. Jede Aufgabe bringt dich einen Schritt näher zum Experten in Spring Boot!

Spielanleitung

  1. Beginne mit der ersten Aufgabe, indem du auf "Spiel starten" klickst.
  2. In jeder Aufgabe erhältst du eine Beschreibung, was du tun musst.
  3. Schreibe deinen Code direkt im integrierten Editor und teste ihn live.
  4. Erfülle die Aufgabe, um zum nächsten Level zu gelangen.
  5. Wiederhole die Schritte und werde ein Spring Boot Experte!
